Useful Information

  1. What is a target (or deliverable)? A target is the final build artifact that is either a library or a executable. In CMake, the targets are defined by add_executable or add_library commands. To read more, click here
  2. How can I find the targets for a project? Each CMake project starts with CMakeLists.txt file in the root folder. You can start finding target related commands in this file for each project.
  3. Can I use my own IDE or other tools? Yes!
  4. Do I have access to the project source code? We will give you the repository URL of the project in each step.
