How to use the tool (DiPiDi)

A tool called DiPiDi will be provided to you during the tasks. DiPiDi can help you to complete the tasks by revealing the exposure of a change. Here is how to use the tool:

  1. Select the file you are interested to see the impact
  2. You can see the impacted targets and the conditions under which the file will impact the target
  3. Additionally you can see the impacted targets given a list of conditions
