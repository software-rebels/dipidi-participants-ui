How to use the tool (DiPiDi)

To assess with your tasks, you may use the tool provided to you called DiPiDi. DiPiDi can help you to complete the tasks by revealing the exposure of a change. Here is how to use the tool:

  1. Select the file you are interested to see the impact. Alternatively, you can select a commit and the tool will show you all the impacted targets because of the changes in that commit.
  2. You can see the impacted targets and the conditions under which the file will impact the target
  3. Additionally you can see the impacted targets given a list of conditions

A condition can be a boolean (BoolRef), a number (ArithRef) or a string (SeqRed). Choose the right type accordingly.

If you faced any problem using the tool, feel free to refresh the page.
